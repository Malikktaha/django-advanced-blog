# Advanced Blog

## Description
Advanced Blog is a fully functional blogging web application built with Django. It allows users to create, edit, and delete blog posts, comment on posts, and explore categories and tags. The project demonstrates user authentication, database relationships, and a modern web interface.

## Features
- User authentication: Sign up, login, logout
- Create, edit, and delete blog posts
- Add and manage categories and tags
- Commenting system for posts
- Pagination for blog listing
- Search functionality
- Admin panel to manage users, posts, and comments
- Responsive design with HTML/CSS

## Tech Stack
- Python 3
- Django 4.x
- SQLite
- HTML, CSS, Bootstrap (optional if used)
- JavaScript (optional if used)
